# Журнал изменений

Все значимые изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [семантического версионирования](https://semver.org/spec/v2.0.0.html).

## [Не выпущено]

### Добавлено
- **НОВОЕ**: Комплексная система автоматических обновлений с интеграцией GitHub Releases
- **НОВОЕ**: Умный импорт инстансов с режимом слияния для добавления только отсутствующих файлов
- UI уведомления об обновлениях с индикатором прогресса скачивания и отображением changelog
- CLI команды для управления обновлениями: `qm update check`, `qm update download`, `qm update info`
- Автоматическая замена бинарных файлов и перезапуск приложения после обновления
- Кроссплатформенная поддержка обновлений для Windows, macOS и Linux
- Расширенная функциональность импорта инстансов с флагом `--merge`

### Изменено
- **BREAKING**: Удалена поддержка немецкого языка, русский установлен как язык по умолчанию
- Упрощена система локализации для поддержки только английского и русского языков
- Русский язык теперь является основным языком с английским как запасным вариантом
- Улучшенные сообщения об ошибках импорта с понятными подсказками для пользователей
- Кроссплатформенная нормализация путей в импортированных файлах инстансов (Windows/Linux)
- Автоматическое преобразование обратных слэшей в прямые в текстовых конфигурационных файлах при импорте
- Полная конфигурация TypeScript со строгой проверкой типов
- Кастомные хуки для Wails runtime и коммуникации с бэкендом
- React Context провайдеры для темы и состояния приложения
- Комплексные определения типов для интеграции с Wails
- Улучшенный рабочий процесс разработки с инструментами TypeScript

### Изменено
- **BREAKING**: Миграция фронтенда с Vue.js на React с полной поддержкой TypeScript
- Замена shadcn-vue на shadcn/ui компоненты
- Обновлена система сборки для использования Vite с React вместо Vue
- Улучшена типобезопасность с комплексными определениями TypeScript
- Добавлены кастомные хуки для интеграции с Wails бэкендом
- Реализован React Context для управления глобальным состоянием
- Улучшены команды импорта инстансов с лучшей обработкой ошибок и подсказками пользователям

- Бэкенд на Go с базовой структурой приложения

- Среда разработки с горячей перезагрузкой
- Базовые UI компоненты и стилизация
- Полноценный Makefile с командами сборки, разработки и контроля качества
- Линтинг Go с golangci-lint и инструментами форматирования кода
- Линтинг фронтенда с ESLint и Prettier
- Обновленный .gitignore с полными исключениями
- Улучшенная система кроссплатформенной сборки с командами для конкретных архитектур
- Автоматическое определение версии и именование сборок
- Удобные цели make для сборки под Linux, macOS и Windows
- Исправлена проблема дублирования директорий сборки (build/bin/build/...)
- Добавлены проверки кросскомпиляции для неподдерживаемых комбинаций платформ
- Улучшена обработка выходных файлов сборки с перемещением после сборки
- Интегрирована библиотека UI компонентов shadcn-vue
- Добавлен Tailwind CSS для современного стилизации
- Создан базовый набор UI компонентов (Button, Card, Input, Label)
- Обновлено основное приложение с демонстрацией новых компонентов
- Добавлена полная поддержка TypeScript для Vue компонентов
- Настроен TypeScript с правильной интеграцией Vue 3
- Добавлена проверка типов и поддержка IntelliSense
- Обновлена система сборки для использования встроенной поддержки TypeScript в Vite
- Исправлена конфигурация PostCSS для совместимости с ES модулями
- Обновлен Vite до v4.5.14 и плагин Vue до v4.6.2
- Решены проблемы компиляции vue-tsc путем использования обработки TypeScript в Vite
- Интегрирована функциональность QMLauncher CLI для функций лаунчера Minecraft
- Добавлен флаг --no-gui для работы в режиме командной строки
- Импортированы CLI команды: start, instance, auth, search, completions, about
- Сохранена обратная совместимость с GUI режимом по умолчанию
- Добавлены GitHub Actions CI/CD workflows для автоматизированной сборки и релизов
- Настроена кроссплатформенная сборка для Linux, macOS и Windows
- Интегрирован процесс сборки frontend в CI пайплайн
- Добавлена автоматизированная загрузка артефактов релиза
- Создана русская версия API документации (API_ru.md)
- Добавлена колонка пути к инстансу в вывод команды `instance list`
- Добавлена команда `java list` для отображения всех установленных версий Java
- Изменена рабочая директория по умолчанию с ~/.minecraft на ~/.qmlauncher
- Реализована UUID-based изоляция инстансов с уникальными директориями

- Модифицированы функции управления инстансами для работы с новой структурой
- Добавлена функциональность экспорта/импорта инстансов
- Добавлена команда 'instance export' для экспорта инстансов в ZIP архивы

- Улучшена проверка исполняемости Java для кроссплатформенной совместимости
- Добавлена опция NoJavaWindow для параметров запуска
