# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-04

### Added
- Initial release of QMLauncher CLI
- Cross-platform Minecraft launcher with mod support
- Command-line interface for managing Minecraft instances
- Authentication and session management
- Java runtime detection and management
- Version and mod loader discovery
- Cross-platform builds with embedded icons
- Automated CI/CD with semantic release
