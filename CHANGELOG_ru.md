## [1.0.0] - 2025-12-27

### Добавлено
- Первоначальная настройка проекта с Wails v2.11.0
- Фронтенд на Vue.js 3 с системой сборки Vite
- Бэкенд на Go с базовой структурой приложения
- Поддержка кроссплатформенного десктопного приложения
- Среда разработки с горячей перезагрузкой
- Базовые UI компоненты и стилизация
- Полноценный Makefile с командами сборки, разработки и контроля качества
- Линтинг Go с golangci-lint и инструментами форматирования кода
- Линтинг фронтенда с ESLint и Prettier
- Обновленный .gitignore с полными исключениями
- Улучшенная система кроссплатформенной сборки с командами для конкретных архитектур
- Автоматическое определение версии и именование сборок
- Удобные цели make для сборки под Linux, macOS и Windows
- Исправлена проблема дублирования директорий сборки (build/bin/build/...)
- Добавлены проверки кросскомпиляции для неподдерживаемых комбинаций платформ
- Улучшена обработка выходных файлов сборки с перемещением после сборки
- Интегрирована библиотека UI компонентов shadcn-vue
- Добавлен Tailwind CSS для современного стилизации
- Создан базовый набор UI компонентов (Button, Card, Input, Label)
- Обновлено основное приложение с демонстрацией новых компонентов
