## 2025-11-29

### Добавлено
- Система зашифрованного конфига с шифрованием AES-256-GCM
- Автоматическая синхронизация с QMServer при первом запуске
- Определение необходимости обновления конфига и автоматическая синхронизация
- Менеджер конфига для хранения зашифрованных данных

### Изменено
- Заменена база данных SQLite на зашифрованный JSON конфиг
- Удалена поддержка плагинов (лаунчер теперь работает только с модами)
- Данные конфига теперь получаются с QMServer и кэшируются локально в зашифрованном виде
- Настройки лаунчера хранятся в зашифрованном config.json

### Удалено
- База данных SQLite (зависимость better-sqlite3)
- Функциональность управления плагинами
- Прямые API вызовы для данных серверов/модов (теперь используется кэшированный зашифрованный конфиг)
